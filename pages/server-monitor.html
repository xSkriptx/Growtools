About Server Monitoring

This tool monitors the official Growtopia servers by checking the game's status API. It provides real-time information about server status, player count, and maintenance periods.

Status Indicators

Online
Server is fully operational with normal player capacity
Maintenance
Server is undergoing maintenance or has limited availability
Offline
Server is completely offline or unavailable
