How It Works

The application generates cache IDs based on date patterns and checks them against Ubisoft's CDN. Valid URLs typically indicate available game updates or assets.

When new cache IDs are discovered, the application sends notifications through Discord, making it easy to stay updated with the latest game changes.

  • Stable internet connection

About Cache Checking

Cache IDs are unique identifiers used by Growtopia to manage game assets and updates. This tool helps you find valid cache URLs by generating and testing various ID combinations.
